Proyecto Pagina web de ecomerce para el condominio CLF
Condominio Camino Las Flores – Plataforma Web

Proyecto web para que los residentes del Condominio Camino Las Flores puedan pagar gastos comunes, revisar información relevante y agendar el uso de instalaciones como la sede o la parrilla.

Este proyecto fue desarrollado utilizando:
HTML5, 
CSS3,
Bootstrap 5, 
JavaScript,
Git & GitHub

Funcionalidades principales:

 Pago de gastos comunes mediante una interfaz simple.
 Carrito de compras usando JavaScript (con contador, sumar/restar/eliminar).
 Agendamiento de la sede y visualización de las fechas ocupadas.
 Secciones informativas sobre el condominio.
 Navbar responsiva con Bootstrap.
 Galería tipo carousel con imágenes del condominio.
 Offcanvas para mostrar el carrito de compras.

El sitio es completamente responsive
